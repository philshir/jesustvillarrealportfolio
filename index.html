Project 1: Reduxtraveler Store

  • Fully functional E-commerce store includes shopping cart, processing credit/debit card payments, user login and registration, and search functionality.
  • Administration functionality to review and manage orders, products, payments and users.
  • Authentication and authorization using JWT; Password encription using Bcrypt.js
  • Built using React for frontend functionality State Management created using Redux
  • Formating and layout using React Bootstrap and React Router
  • Backend functionality using Node.js and Express.js; Non-relational database created with MongoDB, MongoDB Atlas, MongoDB Compass and Mongoose; Deployment using Heroku

I created an e-commerce solution that leveraged the performance and efficiency of the MERN stack. Using React allows a faster build process by being able to reuse components. Redux allows easier state management and less code that is easier to maintain and troubleshoot. Using Express.js and Node.js alows efficient and seamless communication between the front and backend and MongoDB provides a seamless and persistent storage solution.

Technologies used- MERN stack: MongoDB, Express.js, React, Node.js, Redux, Thunk, React Bootstrap, React Router, JWT, bcrypt.js, Heroku , Git, GitHub .

Project 2: Reduxtraveler.com

  • Travel website covering over 450 locations in the United States including maps, itineraries, highlights, hours, seasons, prices, hiking trail details, directions and pictures.
  • Fully responsive for all screen sizes.
  • All graphics, maps and SVGs created using Adobe Illustrator and photos edited and optimized using Adobe Photoshop

I frequently travel and hike but when I am planning trips I often have to go to 3 or 4 different websites to find neccesary details as to how many miles a trail is, is there an entrance fee or if the location is pet friendly. I built this site in order to not only display my graphic design skills and UX/UI, HTML, CSS and Javascript skills but to also provide detail information for all locations I have visited which includes over 25 states and 500 locations. The site is fully responsive and tested on multiple browsers and devices.
